# Development - Agents

❗ Caution! The agent types below are experimental! These are meant as inspiration for your own agents or for further evaluation and development.
❗ Use them only with audiences who know they are engaging an experimental agent!

They become available when enabled using the `ENABLE_DEVELOPMENT_AGENTS` environment variable.

✅ **Civility Agent** - Preemptively intercepts toxic messages and suggests modifications

✅ **Delegates Agent** - Creates a bots-only conversation between agents acting as delegates for participants and an "expert" moderator that uses RAG with topical reference material

✅ **Experts Agent** - Creates a bots-only conversation between agents acting as experts with opposing viewpoints, using RAG with topical reference material

✅ **Playful Agent** - Injects playful reactions and questions into a conversation

✅ **Radical Empathy Agent** - Suggests modifications to messages to promote radical empathy among participants

✅ **Reflection Agent** - Helps participants reflect on areas of agreement and disagreement

[ ] **Agenda Assistant** - Helps a user to prepare a thoughtful, structured agenda conversationally
